Interactive Architecture Diagram

This is an interactive visualization of a system architecture diagram. You can:

  • Press Shift + spacebar to enable interactive mode
  • Then, use arrow keys to move through the sequence step by step
  • Press 'Shift + S' to view the message schemas between components
  • Press 'Shift + L' to reload the diagram after changing the plantUML text
  • Press 'Shift + D' to debug component positioning
